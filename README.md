만약 가정에서 발생하는 대기전력을 절약하여 에너지 취약계층에게 기부할 수 있는 서비스가 있다면?

당기요는 우리집에서 낭비되는 대기전력을 절약하여 에너지 취약계층 이웃들에게 나눌 수 있는 기부 플랫폼입니다.
